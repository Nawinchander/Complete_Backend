Full Backend System (Node.js + MongoDB + MySQL + Docker + Jenkins + AWS)

A complete backend project to understand real-world server architecture using Node.js and Express.
Supports MongoDB (Mongoose) and MySQL for multi-database operations.
Includes middleware, MVC structure, and clean API routing.
Automated builds using Gulp for optimized workflow.
Containerized with Docker for consistent deployment.
CI/CD pipeline implemented using Jenkins.
Production-ready deployment on AWS EC2.
Environment-based configuration using .env.
Designed to learn the full backend flow from API to cloud.
Perfect for backend practice, DevOps learning, and portfolio projects.
